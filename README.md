# Cypher - Real-Time Chat Application

Cypher is a full-stack real-time messaging platform that allows users to communicate through private one-to-one conversations. It provides instant message delivery, secure authentication, persistent chat history, typing indicators, and read receipts.

Built using React.js, FastAPI, MongoDB, and WebSockets.

## Features

- Secure user authentication using JWT
- Password hashing using bcrypt
- Real-time messaging using WebSockets
- Persistent message storage
- One-to-one private conversations
- User search functionality
- Typing indicators
- Message read receipts
- Emoji support
- Responsive user interface

## Implementation Highlights

- Developed REST APIs for authentication, user management, chat creation, and message handling.
- Implemented WebSocket based architecture for real-time bidirectional communication.
- Used asynchronous database operations with FastAPI and Motor for efficient data handling.
- Integrated JWT authentication to secure protected routes and user sessions.
- Managed production deployment using environment variables on Render.
