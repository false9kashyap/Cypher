# CYPHER 

CYPHER is a realtime chat application built using React, FastAPI and MongoDB.

## Features 

- User Registration & Login
- JWT Authentication
- Password Hashing
- User Search
- Private One-to-One Chats
- Realtime Messaging using WebSocket
- Typing Indicator
- Message Read Receipts
- Emoji Support
- User Profile Page
- Responsive UI

## Tech Stack 

### Frontend
- React
- React Router
- Axios
- CSS

### Backend
- FastAPI
- Python
- WebSocket
- JWT Authentication

### Database
- MongoDB
